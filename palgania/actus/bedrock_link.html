Lier vos comptes Java et Bedrock

Notre serveur dispose d'un plugin permettant l'accès à tous les joueurs, qu'ils soient sur Java ou sur Bedrock Edition.
Pour rejoindre le serveur depuis Minecraft Bedrock (mobiles, Windows 10 Edition, consoles, etc.), vous pouvez suivre la procédure ici.

Ce plugin permet aux joueurs possédant une édition Bedrock et ne possédant par l'édition Java de rejoindre le serveur. Cependant, il n'avait jusqu'à présent pas d'utilité pour les joueurs Java, puisqu'en rejoignant le serveur depuis une Bedrock edition, ils se trouvaient sur un compte différent.

Nous avons ainsi décidé d'implémenter un système permettant de lier les comptes Java et Bedrock. Cela permettrait aux joueurs ayant à la fois la Java Edition et la Bedrock Edition, de jouer sur le même compte sur chaque jeu. Par exemple, vous pourriez commencer une partie sur votre PC avec la Java Edition et la reprendre plus tard sur votre téléphone portable, en conservant votre inventaire et toutes vos actions !
Si vous souhaitez bénéficier de cette fonctionnalité, effectuez la procédure suivante !

La procédure

Par défaut, lorsque vous vous connectez sur le serveur depuis une version Bedrock, vous utilisez un compte différent que votre compte Java.
Pour lier ce compte à votre compte Java, il faut tout d'abord connecter vos deux comptes (Java et Bedrock) sur le serveur simultanément.
Ensuite, depuis votre version Java, entrez la commande /linkaccount (NomJoueur) où (NomJoueur) est votre pseudo Bedrock edition.
NB: Les pseudos des joueurs se connectant depuis une version Bedrock sont précédés d'une astérisque (*)
Après avoir tapé la commande, vous obtiendrez un message vous demandant d'exécuter la commande linkaccount sur Bedrock.
Exécutez la commande qui vous est donnée. Votre compte Bedrock sera alors ejecté du serveur. Lors de votre reconnexion, vous serez authentifié avec votre compte Java.

Pour délier vos comptes, tapez la commande /unlinkaccount depuis Bedrock !

Et voilà ! Vous pouvez jouer sur bedrock et sur Java en conservant la même partie !
