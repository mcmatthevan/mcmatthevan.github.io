Lier les comptes Java et Bedrock.

Vous savez maintenant que vous pouvez rejoindre le serveur depuis une version Minecraft Bedrock !
Nous avons désormais implémenté une fonctionnalité permettant de lier les comptes Bedrock et Java, et ainsi de suivre, par exemple, la même partie sur votre PC et votre téléphone !
Pour en savoir plus et connaître la procédure, cliquez ici !

Accessibilité pour les versions Minecraft Bedrock

Notre serveur Minecraft Java Edition est désormais joignable par les joueurs jouant sur Minecraft Bedrock (mobile, console , édition Windows 10, etc.) !
Pour savoir quelle est la procédure et les manipulations spécifiques, cliquez ici !
