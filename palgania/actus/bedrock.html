Accessibilité pour les versions Minecraft Bedrock

Le serveur est accessible pour les joueurs jouant sur des versions Minecraft Bedrock (consoles, tablettes, mobiles). 
L'ip du serveur est 51.77.148.210 est le port est celui par défaut (19132). 

Pour les consoles, une manipulation est nécessaire pour permettre l'accès à des serveurs externes. 
Voici comment faire pour chaque type de console :

Pour Xbox :

Pour commencer, rendez-vous dans les paramètres de la console. Ensuite allez dans les paramètres réseaux. 
Une fois dans les paramètres réseaux, allez dans "paramètres avancés" puis dans "paramètres DNS". Sélectionnez "Manuel" et entrez comme DNS primaire :
104.238.130.180
Puis comme DNS secondaire :
8.8.8.8
Une fois ces paramètres modifiés et appliqués, lancez votre Minecraft. Rendez-vous dans la liste des serveurs ( pas realm attention ) et cliquez sur n'importe lequel. 
Après le chargement, vous devriez arriver sur une page "server list" ou quelque chose du genre. 
Sélectionnez ensuite "connect to a server" et entrez dans le champ "server adress" l'ip de notre serveur : 51.77.148.210 et dans le champ "Server port" entrez "19132"
Vous pouvez alors vous connectez à notre magnifique serveur !

Pour Nintendo Switch :

Pour commencer, rendez vous dans les paramètres votre Switch. Ensuite allez dans les paramètres Internet.
Une fois dans les paramètres Internet, allez dans "paramètres avancés" puis Sélectionnez votre Wifi et cliquez sur "changer les paramètres". Rendez-vous dans la section "paramètres DNS" puis sélectionnez "manuel". Vous pouvez alors entrez comme DNS primaire :
104.238.130.180
Puis comme DNS secondaire :
8.8.8.8
Une fois ces paramètres modifiés et appliqués, lancez votre Minecraft. Rendez-vous dans la liste des serveurs ( pas realm attention ) et cliquez sur n'importe le quel. 
Après le chargement, vous devriez arriver sur une page "server list" ou quelque chose du genre. 
Séléctionez ensuite "connect to a server" et entrez dans le champ "server adress" l'ip de notre serveur : 51.77.148.210 et dans le champ "Server port" entrez "19132"
Vous pouvez alors vous connectez à notre magnifique serveur !
